# Reglamento de Star Wars: Destiny — referencia interna

Referencia **completa y reorganizada con nuestras palabras** de las reglas de Star Wars: Destiny,
para consulta durante specs e implementación. **No es una copia** del documento oficial: es un
resumen/paráfrasis exhaustivo, con punteros de página al *Rules Reference v1.07.01* (ARH CC).

- Fuente oficial: *Star Wars: Destiny Rules Reference v1.07.01*, A Renewed Hope Continuing
  Committee — https://swdrenewedhope.com
- El **PDF no se sube** al repo (copyright; repo público). Este resumen sí, por ser obra propia.
- **Regla del oro** (RR pg 3): si el texto de una carta contradice las reglas, manda la carta.
- Para este proyecto, ante conflicto entre reglamento y GDD, **manda el GDD** y se PARA a preguntar
  (regla 1 de `CLAUDE.md`). Este reglamento es material de consulta, no cambia specs por sí solo.
